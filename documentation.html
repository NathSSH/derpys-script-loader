derpy's script loader 9

Visit the full documentation for derpy's script loader (DSL) on bullyscripting.net. This light documentation lists most functions, explains what is included with this release, and covers some of the basics you'll need to get started.

_derpy_script_loader/scripts

This is the most important folder, since it is where all your script collections go. You can think of these collections as individual mods, but really they are just a way of grouping related scripts together. There are 3 main types of collections.

  • NORMAL: a folder is the most common type of script collection.
  • ZIPPED: a zip file structured like a normal collection. The root of the zip is just treated like a folder.
  • STANDALONE: single scripts (not in a folder) are still counted as collections, just containing themselves.

It is really important when installing mods that you do not put files from a non-standalone collection directly into scripts. If you do, then each individual script will be treated like its own collection and start automatically (which may not be the author's intention).

The only script included by default is loadanim.lua, so that other scripts don't need to worry about it.

_derpy_script_loader/extras

This folder contains a few optional parts of DSL. If you want to use any of the scripts here, move them to your scripts folder (just remember to move the whole folder and not just its files).

  • dev: scripts for helping developers. See the comments in each script for more help.
  • sdk: files for extending DSL using packages. Not of interest to 99% of people.
  • window: scripts to run the game in windowed mode. See the config file for more settings.

_derpy_script_loader/examples

This folder has example scripts to help you get started or learn more. All of them require you to know the basics of Lua 5.0.2. If you don't know Lua, consider reading chapter 1 of Programming in Lua while you look at some of the starter examples.

There are 4 "difficulty levels" of example. Start wherever you think is appropriate for you and move up. All of them have their own readme.txt file that summarizes the example and offers some challenges for you.

STARTER

Start learning how to structure script collections, make scripts, and use the console.

  • hello_world: show "Hello, world!" on the screen.
  • give_cash: give the player money when using /give_cash in the console.

BEGINNER

See how you can do a bit more while still having lots of comments to guide you.

  • free_stuff: make all items in shops free.
  • trash_loot: loot any trash can for random stuff once a day.
  • spawn_bodyguard: spawn a bodyguard using /bodyguard in the console.

INTERMEDIATE

Start looking at more complex examples since you already know the basics.

  • custom_healthbar: draw a custom healthbar using a texture.
  • riot_mode: make all peds fight each other and spawn more peds.

EXPERT

Make use of more DSL features to make your scripts even better.

  • algie_survival: survive against waves of Algernon.
  • mod_menu: a menu that lets you control a bunch of things.
  • super_nerds: all the nerds are a lot stronger, super even.
