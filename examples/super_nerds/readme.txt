derpy's script loader: super_nerds example

difficulty:
 this is an EXPERT example, comments are less verbose and so they can focus on more complex concepts

prerequisites:
 you should be familiar with using Lua, making script mods, and with the concept of threads

summary:
 nerds are better fighters, with better stats and custom fight moves

files:
 config.txt: the config for the script collection
 nerds.lua: the main script

challenges:
 1. make nerds do a custom attack when they're mounting somebody (grappling them on the ground)
 2. make nerds use customly made combos, where you specify different attacks in a chain
